📒 내 메뉴

한 번 저장하면 이 폰에 계속 저장돼요. 키는 이 기기 밖으로 나가지 않아요(구글 API 호출에만 사용).
발급: aistudio.google.com → Get API key (무료)

📚 출처 라이브러리

💡 정확도 팁: 채널 메인보다 특정 영상·레시피 URL을 넣으면 AI가 그 내용을 직접 읽어요.
🧠 AI 지시문 커스텀
추천할 때 AI에게 보내는 지시문이에요. 기본값이 있고, 원하는 대로 다듬을 수 있어요. 수정한 것도 폰에 저장돼요.
🧹 데이터
모든 데이터(키·재료·메뉴·출처)는 이 폰 브라우저에만 저장돼요. 브라우저 데이터를 지우면 함께 지워져요.
